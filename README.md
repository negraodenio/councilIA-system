# CouncilIA — Multi-Agent Decision System

AI system that simulates structured debate between expert agents to produce better decisions.

---

## 🚨 Problem

Most AI systems agree with the user.

- High bias  
- Low reliability  
- Weak decision quality  

And in companies:

- Decisions take too long  
- Opinions are unstructured  
- Critical risks are missed  

---

## 💡 Solution

CouncilIA uses multiple AI agents with different perspectives to debate an idea before producing a final recommendation.

Instead of a single answer, it delivers:

- structured analysis  
- opposing viewpoints  
- a final, reasoned decision  

---

## ⚙️ How it works

### Round 1 — Independent Analysis
Each agent analyzes the problem separately.

→ No bias, no influence  

---

### Round 2 — Adversarial Debate
Agents challenge each other’s assumptions.

→ Stress-testing ideas  

---

### Round 3 — Final Decision
A judge aggregates the debate into a final recommendation.

→ Clear, structured output
